## Run it localy: 

- clone the repo: 
 
```bash
git clone https://github.com/AM-77/rigel.git
```

- install the dependencies: inside the `rigel` folder

```bash 
npm install
# or
yarn
```

- run the app (with its default data):

```bash
npm run start
# or
yarn start
```

- in your browser, open a window on `http://localhost:3000` 🎆
